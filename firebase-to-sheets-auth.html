📋 วิธีใช้งาน

  1. 1.
    Login Firebase

    ใส่ Email และ Password จาก Firebase

  2. 2.
    กด "โหลดจาก Firebase"

    รอสักครู่ให้โหลดข้อมูลทั้งหมด

  3. 3.
    ตรวจสอบ Preview

    ดูข้อมูลที่โหลดมาว่าถูกต้องหรือไม่

  4. 4.
    กด "Export เป็น CSV"

    ดาวน์โหลดไฟล์ CSV

  5. 5.
    Import ไป Google Sheets

    File → Import → เลือกไฟล์ CSV
