📄 CSV Output

📋 วิธีนำไป Google Sheets

  1. 1.
    แปลง JSON เป็น CSV

    กด "แปลงเป็น CSV" หรือวาง JSON แล้วกดปุ่ม

  2. 2.
    Preview CSV

    กด "Preview CSV" เพื่อดูข้อมูลก่อน

  3. 3.
    ดาวน์โหลด CSV

    กด "ดาวน์โหลด CSV" หรือ "Copy CSV ไป Clipboard"

  4. 4.
    Import ไป Google Sheets

    เข้า Google Sheets → File → Import → เลือกไฟล์ CSV

💡 เคล็ดลับ

  • • หลัง Import ตรวจสอบว่าข้อมูลครบทุก field
  • • Checklist จะแสดงเป็น JSON string เช่น {"signal":true,"sl":true}
  • • ถ้าไม่ถูกใช้ Google Sheets Import อีกครั้ง
  • • ถ้ายังไม่ได้ ใช้วิธี Copy ไปวางใน Google Sheets
